Welcome to **Journal-ing**! It's where you Journal your life.

Journal-ing is a small blog-like Web Application where you can keep track of your day and your work. This small Web application was meant to be used mainly by developers to document their daily work and their project's progression over time (just like a "commit changes" button in Github).

In order to use this small web application:
1. Use the "clone" command followed by the project's repository: ```https://github.com/Ivan-Tapia-Git/journaling-project-app.git```. Or you can download and unzip the ZIP file from the Github repository.
2. Inside the project's folder, open a terminal window. (Or you can open a terminal window first and then navigate to the project's folder)
3. In the terminal, create a virtual environment using Python with the following command: (Mac/Linux) ```python -m venv <name_of_virtual_env>``` | (Windows) ```python3 -m venv <name_of_virtual_env>```
4. Activate the Virtual Environment using: (Mac/Linux) ```source <name_of_virtual_env>/bin/activate``` | (Windows) ```./<name_of_virtual_env>/Scripts/Activate```

(You should now see the name of your virtual environment inside parenthesis before the terminal statement)

5. Install the requiered packages using: (Windows/Mac/Linux) ```pip3 install django```
6. Start and run a local server using: (Mac/Linux): ```python manage.py runserver <port_number>``` | (Windows) ```python3 manage.py runserver <port_number>```
7. Open the link provided by the terminal on your prefered Web Browser. (e.g "http://127.0.0.1:8000/")
8. Start using Journal-ing!

Journal-ing is a personal project I have been working on since August of 2025. My goal with this project is to learn and master Back-End Tools to develop a career in Software Engineering.
This project was build using the Django Web Framework.
